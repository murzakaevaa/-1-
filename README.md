1. Форматирование и стиль кода
   
Отступы: исправлены с 4 пробелов на 2 пробела в соответствии с Google Style Guide

Положение скобок: открывающие скобки { перенесены на ту же строку, закрывающие } на отдельной строке

Имена переменных: изменены с сокращенных на понятные snake_case:

  n → item_count (количество предметов)

  maxWeight → max_total_weight (максимальный вес)

  dp → min_tax (минимальный налог для веса)

  selected → is_selected (матрица выбранных предметов)

  minVal → best_min_tax (лучший минимальный налог)

  bestWeight → best_weight (лучший вес)

  bestSelection → best_selection (лучший набор)

Имена функций: knapSack → find_optimal_artifacts (lower_case_with_underscores)
